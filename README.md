 wanted to create a website that looks and feels like a proper e‑commerce experience – but without the backend complexity. So I built a multi‑page static site with:

> A Homepage that introduces the brand and links to product categories.
> Separate pages for Earbuds, Headphones, and Speakers, each with product cards and a “Buy Now” button that opens a dynamic modal with the correct product info.
> A Features page that highlights the tech (noise cancellation, battery life, etc.).
> A Gallery with clickable tiles that show multiple images and key details in a modal.
> A Reviews page with real‑sounding testimonials from folks named Varshini, Deepa, Ashwini and Vijayendra.
> An FAQ page with common questions in a neat accordion layout.

Every page shares the same navigation bar and footer, so the whole thing feels consistent. There’s even a handy “Back to Home” button on every inner page – because nobody likes getting lost.
